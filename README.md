# Comparativa MacBook Air M1 vs M2

Este proyecto es una página interactiva en Vue 3 para comparar dos modelos de MacBook Air, enfocándose en las diferencias de rendimiento para diseño y uso general.

## Instrucciones para ejecutar

1.  Abre una terminal en esta carpeta.
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Ejecuta el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Abre el link que aparece en la terminal (usualmente `http://localhost:5173`).

## Características

-   **Comparativa Interactiva**: Selecciona tu perfil de uso (General, Diseño, Estudiante) para ver recomendaciones personalizadas.
-   **Enfoque en RAM**: Explicación detallada de por qué 8GB vs 16GB importa para diseño.
-   **Diseño Responsive**: Se adapta a móviles y escritorio.
-   **Datos Reales**: Basado en los links proporcionados de Paris.cl y Tecnomas.cl.
